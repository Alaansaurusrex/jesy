# EcoClima Kids

EcoClima Kids es una aplicación web educativa para que niños y niñas exploren el clima de diferentes ciudades y aprendan sobre la importancia de cuidar el planeta.

## Características
- Consulta el clima actual de cualquier ciudad.
- Recomendaciones ecológicas según el clima.
- Interfaz colorida y amigable para niños.
- Animaciones y efectos visuales divertidos.

## Instalación y uso
1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web.
3. Ingresa el nombre de una ciudad y haz clic en "¡Buscar!" para ver el clima.

## Tecnologías utilizadas
- HTML5
- CSS3 (Bootstrap y estilos personalizados)
- JavaScript
- Font Awesome (iconos)
- OpenWeatherMap API
